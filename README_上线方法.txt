上线到 GitHub Pages：
1. 创建一个新的公开 GitHub repository。
2. 将本文件夹内的5个文件上传到仓库根目录。
3. 仓库 Settings → Pages。
4. Build and deployment 选择 Deploy from a branch。
5. Branch 选择 main，文件夹选择 /(root)，保存。
6. 等待页面生成后，用显示的 https://用户名.github.io/仓库名/ 地址访问。
7. iPhone/iPad：用 Safari 打开 → 分享 → 添加到主屏幕。

注意：
记录存储在每台设备自己的浏览器中，并不会自动跨设备同步。
请定期导出 CSV 或 JSON 备份。
