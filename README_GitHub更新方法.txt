公開版：個人情報を含みません。

GitHub の leg15-app に、フォルダ内の以下のファイルをすべてアップロードしてください。
・index.html
・manifest.webmanifest
・sw.js
・icon-192.png
・icon-512.png
・README_GitHub更新方法.txt

使い方：
1. Safariで公開ページを開く
2. 「端末のExcelを読み込む」を押す
3. iPhoneの「ファイル」等に保存した .xlsx / .xls を選ぶ
4. 編集後、「全データをExcelに保存」で最新版を端末へ保存する

Excelや氏名データはGitHubへアップロードしないでください。
更新後、古い画面が出る場合はURL末尾を変更して開いてください。
例：https://theaooranger-hue.github.io/leg15-app/?v=6
